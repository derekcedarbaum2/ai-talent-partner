## Patterns that worked (harvest from the bank)
- Open with a quantified credential that also names the candidate's contribution ("I was the product lead for the prototype that proved X, and that proof unlocked $120M+").
- Make the role's core competency the spine with one challenge to action to result story (a hardware-defect to re-sourcing-a-manufacturer story for a Quality role).
- Mission close in the candidate's own words, backed by a demonstrated prior choice. Never quote the company's tagline.
- Differentiator equals a built thing with a mechanism, not a usage stat ("I built my team's memory system; frontier labs don't give you that out of the box").
